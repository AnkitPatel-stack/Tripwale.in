# TripWale.in - Travel Booking Platform

A modern travel booking website built with React, Material UI, and Three.js for immersive travel experiences.

## Features

- 🎨 Modern UI with Material UI
- 🌍 3D visualizations with Three.js
- 📱 Fully responsive design
- 🔍 Advanced search and filtering
- 🎯 Component-based architecture
- ⚡ Fast performance with Vite

## Tech Stack

- **Frontend**: React 18, Material UI, Three.js
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **HTTP Client**: Axios
- **Animations**: Framer Motion
